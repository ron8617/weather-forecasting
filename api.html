Overview

The Weather API provides developers with access to global weather data, including current conditions, forecasts, historical data, and air quality information. Our API delivers accurate, reliable weather data sourced from multiple meteorological providers and processed with proprietary algorithms.

Features

  • Real-time weather data for any location
  • 5-day/3-hour forecasts with 99% accuracy
  • Historical weather data archive (10+ years)
  • Air quality index (AQI) measurements
  • Hyperlocal weather predictions
  • Severe weather alerts

Authentication

All API requests require an API key for authentication. Include your API key in the request header:

GET https://api.weatherapp.com/v1/current?city={city} 

GET /v1/current?city=London HTTP/1.1
Host: api.weatherapp.com
Authorization: Bearer YOUR_API_KEY
Accept: application/json

You can get your API key by signing up at developer.weatherapp.com. Free tier includes 1,000 requests/day.

Current Weather

Get current weather data for any location by city name, ZIP code, or geographic coordinates.

GET https://api.weatherapp.com/v1/current?city={city}&units={metric|imperial}

Parameters

Parameter Type Required Description
city string Yes* City name (either city or lat/lon required)
lat number Yes* Latitude (decimal)
lon number Yes* Longitude (decimal)
units string No Unit system (metric or imperial, default: metric)

Response Example

200 Success
400 Error

Forecast

Get weather forecast data with 3-hour intervals for 5 days.

GET https://api.weatherapp.com/v1/forecast?lat={lat}&lon={lon}&units={metric|imperial}
